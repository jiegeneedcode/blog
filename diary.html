Learning Diary

记录 AI Agents 学习的每一天，从基础概念到实战应用，见证成长的每一步

01 Day

初识 AI Agents

2026-04-15
⏱️ 3.5 小时
📚 5 篇文章
💻 2 个Demo
今天正式开启 AI Agents 的学习之旅！从最基础的概念开始，了解什么是 Agent、它与传统 AI 助手的区别、以及为什么 Agent 是 AI 发展的下一个重要方向。通过阅读 OpenAI 的文档和 Andrew Ng 的课程，建立了对 Agent 工作流的基本认知。

💡 核心要点

  • Agent = LLM + 工具调用 + 记忆 + 规划能力
  • ReAct 模式：思考 → 行动 → 观察 → 迭代
  • Agent 擅长处理复杂、多步骤的任务
02 Day

LangChain 快速上手

2026-04-16
⏱️ 5 小时
📚 8 篇文章
💻 4 个Demo
学习了 LangChain 的核心组件：Chains、Agents、Memory。动手实现了第一个带工具调用的 Agent，可以自动搜索网页、计算数学问题。最兴奋的是成功接入了 Claude API，让 Agent 具备了更强大的推理能力。

💡 核心要点

  • LangChain Expression Language (LCEL) 简化链式调用
  • Tool Calling 是 Agent 与外界交互的关键
  • Memory 模块让 Agent 保持对话上下文
03 Day

RAG 技术深入

2026-04-17
⏱️ 6 小时
📚 6 篇文章
💻 3 个Demo
今天深入学习了 RAG（检索增强生成）技术。从文档切分策略、向量数据库选择（ChromaDB vs Pinecone）、到混合检索实现，完整搭建了一个本地知识库问答系统。理解了 Embedding 的本质：把文本转化为向量，在向量空间中捕捉语义相似性。

💡 核心要点

  • Chunk Size 影响检索精度，需要根据场景调优
  • 语义分块比固定长度分块效果更好
  • Reranking 可以进一步提升相关性
04 Day

Multi-Agent 系统

2026-04-18
⏱️ 7 小时
📚 4 篇文章
💻 2 个项目
今天挑战了更复杂的 Multi-Agent 架构！实现了一个由 Planner Agent、Researcher Agent、Coder Agent、Reviewer Agent 组成的协作系统。每个 Agent 有明确的角色和分工，通过消息传递实现协作。第一次感受到"AI 社会"的魅力——不同的 Agent 像不同的专家一样各司其职。

💡 核心要点

  • Agent 间的通信协议设计至关重要
  • 适当的 System Prompt 是定义角色行为的关键
  • 错误处理和重试机制保证系统稳定性
